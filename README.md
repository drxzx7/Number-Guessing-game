# 🎯 Enhanced Number Guessing Game

A feature‑rich Number Guessing Game built with Python Tkinter, featuring a **Power BI‑style analytics dashboard** with live charts, detailed statistics, and persistent data storage.

## ✨ Features
- Three difficulty levels (Easy, Medium, Hard)
- Smart hint system (limited per game)
- Timer and attempt tracking
- Dynamic scoring with bonus/penalty
- Real‑time dashboard with KPIs
- **Visual Analytics**:
  - Wins vs Losses bar chart
  - Score trend line chart
  - Difficulty distribution chart
  - Attempts distribution chart
- Game history saved to JSON
- Advanced stats page with detailed metrics

## 🛠️ Technologies
- Python 3.10+
- Tkinter (GUI)
- JSON (data persistence)
- Random, Time, Collections modules
